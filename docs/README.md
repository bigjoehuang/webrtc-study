## 🔍 快速导航

### 新用户入门
1. 先阅读 [README.md](../README.md) 了解项目概况
2. 查看 [用户指南](./USER_GUIDE.md) 学习如何使用各个示例
3. 遇到问题时查阅用户指南中的 **常见问题** 部分

### 开发者入门
1. 阅读 [开发者指南](./DEVELOPER_GUIDE.md) 搭建开发环境
2. 查看 [架构文档](./ARCHITECTURE.md) 理解系统设计
3. 参考 [API 文档](./API.md) 了解技术细节

### 深入学习
1. 研究 [架构文档](./ARCHITECTURE.md) 中的系统设计图
2. 阅读 [API 文档](./API.md) 掌握 WebRTC 核心技术
3. 查看源代码中的详细注释
